Ejemplos de Jquerry

Hide

Permite ocultar un elemento
(hay una imagen oculta)

Show

Permite mostrar las secciones ocultas de un elemento

Togle

Permite mostrar o ocultar un elemento

slideUp

Oculta un elemento el efecto de deslizamiento (acordeón).

slideDown

Muestra un elemento con un efecto de deslizamiento (acordeón)

slideToggle

Muestra u oculta un elemento con efecto de deslizamiento (acordeón).

fadeIn

muestra un elemento (anteriormente oculto) cambiando su opacidad

fadeOut

asigna un nivel de opacidad de un elemento dado.

addClass

podemos añadir clases a los elementos

before

añade contenido antes del elemento

after

añade contenido despues del elemento

append

añade o inserta contenido al final del elemento seleccionado
